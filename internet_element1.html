Hardware & medium of communication

The internet is completely open, the connections are shared, and information is sent as plain text.The internet elements are classified as hardware, software, protocols and connection medium.There are three layers – wires, cable and Wi-Fi, TCP/IP & ROUTING, HTTPS & DNS works together for seamless data transmission.

Internet ships binary information. Everything in internet is centred around bits. Bits can be “0” or “1”. 8 bits is 1 byte .1000 byte is 1KB, 1000KB is 1 MB.On an average a song can, be 3-4 megabyte.

Early means of communication were done using beeps, morse codes or lights” on or “off”. Bits can be transmitted using three mediums: Electricity, light and radio waves. Bandwidth is measured bitrate (no. of bits per second) and it is the maximum transmission capacity and latency is the time taken for a bit to travel. Cables like ethernet are cheap but have more signal loss. Fibre optic cables are expensive but can be used for long distance communication with no signal loss. They have thread of glass to reflect light. Wireless medium used radio signals with different frequencies to transmit and receiving signals which is then later converted to binary format.

IP Address

The internet is an architecture expressed in well-known standards called as Internet Protocol. All devices have unique number known as IP address. IPv4 is an IP version widely used to identify devices on a network using an addressing system.

It uses a 32-bit address scheme to store 2^32 addresses which is more than 4 billion addresses.IPv6 is the most recent version of the Internet Protocol. This new IP address version is being deployed to fulfil the need for more Internet addresses. It was aimed to resolve issues that are associated with IPv4. With 128-bit address space, it allows 340 undecillion unique address space. DNS and IPs are designed to scale.

DNS

Domain name system-DNS, or the Domain Name System, translates human readable domain names (for example, www.amazon.com) to machine readable IP addresses (for example, 192.0.2.44). The Internet’s DNS system works much like a phone book by managing the mapping between names and numbers.DNS servers translate requests for names into IP addresses, when they type a domain name into their web browser.
