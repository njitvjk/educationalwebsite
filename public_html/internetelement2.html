ICANN

The domain name system is administered by the (ICANN), a non-profit organization based in California. ICANN was founded in 1998. It was granted authority over DNS by the US Commerce Department, though it has increasingly asserted its independence from the US government. There are two types of domain names. The first is generic top-level domains (gTLDs) such as .com, .edu, .org, and .gov. There are also country-code top-level domains (ccTLDs).

Each country in the world has its own 2-letter code(.cn,.us). ICANN is also responsible for distributing IP addresses to ensure that two different organizations don’t use the same address.

W3C

W3C is the standard for the Web and it has many member organizations to help facilitate interoperability among the web users. Incompatible versions of HTML are offered by different vendors, causing inconsistency in how web pages are displayed. The consortium tries to get all those vendors to implement a set of core principles and components that are chosen by the consortium.

When the browsers follow W3C standards , the webpages can be accessed in any different browsers. W3C provides standards for HTML,CSS, Web graphics( audio,image, vedio), scripting and dynamic content. It also provides privacy and security guidelines.

WWW

The World Wide Web or the Web is a collection of document resources available in server are accessed through Uniform Resource Locators (URL example https://xyz.com) over the Internet. These resources are transferred using HTTP or HTTPS and are available for users through the software called as the Web browser. WWW is used by billions of people to access internet.

English scientist Tim Berners-Lee invented the World Wide Web in 1989. He wrote the first web browser in 1990 while employed at CERN near Geneva, Switzerland. The Web was available for everyday use from 1993 and is the key reason for the tremendous growth of internet.
